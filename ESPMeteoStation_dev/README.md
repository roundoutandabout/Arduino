#ESP8266 Weather Station
##Weather Station, that uses:
<ul>
	<li><b>ESP12</b> (I use WIFI WITTY / GIZWITS ESP-12F MODULE)</li>
	<li><b>DHT11/22</b></li>
	<li><b>BMP180</b></li>
	<li>Analog Photoresistor</li>
</ul>
> ###Sending to ThingSpeak & Narodmon
> ###Simple web-interface on client side
